# LangUp.ai 사용자 시나리오

이 문서는 LangUp.ai 서비스의 주요 사용자 시나리오와 사용자 여정을 설명합니다. 각 시나리오는 사용자가 서비스를 통해 경험하는 전형적인 흐름을 보여줍니다.

## 시나리오 1: 신규 사용자 온보딩

### 사용자 프로필
- **이름**: 민지
- **나이**: 27세
- **직업**: 국제 무역 회사 신입 사원
- **목표**: 업무에 필요한 영어 회화 능력 향상

### 사용자 여정

1. **서비스 발견**
   - 민지는 소셜 미디어에서 LangUp.ai 광고를 발견합니다.
   - 홈페이지를 방문하여 서비스 소개와 기능을 탐색합니다.

2. **회원가입**
   - '무료로 시작하기' 버튼을 클릭합니다.
   - 이메일과 비밀번호로 계정을 생성합니다.
   - 개인정보 이용 약관에 동의합니다.

3. **초기 평가**
   - 언어 학습 목표를 선택합니다 (업무용 영어 회화).
   - 현재 영어 수준을 평가하는 간단한 테스트를 완료합니다.
   - 학습 선호도에 관한 질문에 답변합니다 (학습 시간, 방법 등).

4. **맞춤형 학습 계획 생성**
   - AI가 민지의 영어 수준, 목표, 선호도를 분석합니다.
   - 비즈니스 영어에 중점을 둔 맞춤형 학습 경로가 생성됩니다.
   - 시스템이 주 3회, 30분씩 학습할 것을 추천합니다.

5. **첫 학습 세션**
   - 민지는 추천된 첫 번째 레슨을 시작합니다.
   - 비즈니스 미팅에서 사용되는 기본 표현을 학습합니다.
   - 발음 연습을 위해 AI와 대화 연습을 진행합니다.
   - 학습 완료 후 피드백과 다음 학습 추천을 받습니다.

## 시나리오 2: 정기 사용자의 일상적 학습

### 사용자 프로필
- **이름**: 준호
- **나이**: 32세
- **직업**: 소프트웨어 개발자
- **목표**: 일본어 문화와 기술 문서를 이해하기 위한 일본어 능력 향상
- **사용 기간**: 3개월

### 사용자 여정

1. **일일 학습 알림**
   - 준호는 오전 7시에 일일 학습 알림을 받습니다.
   - 모바일 앱을 통해 LangUp.ai에 로그인합니다.

2. **대시보드 확인**
   - 학습 진행 상황 대시보드를 확인합니다 (현재 진도 45%).
   - 어제 학습한 내용의 복습 퀴즈가 추천됩니다.

3. **복습 퀴즈 완료**
   - 어제 배운 일본어 기술 용어에 대한 퀴즈를 풀어봅니다.
   - 몇 가지 틀린 답변에 대해 개인화된 피드백을 받습니다.
   - 시스템이 약점으로 표시된 부분에 대한 추가 연습을 제안합니다.

4. **새로운 학습 콘텐츠**
   - 오늘의 레슨으로 넘어갑니다 (일본 비즈니스 이메일 작성법).
   - 관련 어휘와 표현을 배웁니다.
   - 실제 이메일 예시를 통해 문맥 속에서 표현을 학습합니다.

5. **실전 연습**
   - 일본어로 간단한 업무 이메일을 작성하는 연습을 합니다.
   - AI가 실시간으로 피드백을 제공하고 개선 사항을 제안합니다.
   - 수정된 최종 이메일을 저장하여 나중에 참조할 수 있습니다.

6. **성과 확인**
   - 학습 세션이 끝나고 새로운 포인트와 업적을 획득합니다.
   - 주간 목표 달성 현황을 확인합니다 (5일 중 3일 완료).
   - 다음 학습 세션 일정을 확인합니다.

## 시나리오 3: 프리미엄 기능 업그레이드

### 사용자 프로필
- **이름**: 소희
- **나이**: 35세
- **직업**: 마케팅 매니저
- **목표**: 스페인 지사와의 협업을 위한 스페인어 습득
- **사용 기간**: 1개월 (무료 플랜)

### 사용자 여정

1. **한계 경험**
   - 소희는 한 달간 무료 플랜으로 기본 스페인어를 학습했습니다.
   - 실시간 화상 대화 연습과 같은 일부 고급 기능이 제한되어 있음을 발견합니다.
   - '프리미엄 기능 이용하기' 알림을 확인합니다.

2. **플랜 비교**
   - 프리미엄 플랜 페이지로 이동하여 다양한 구독 옵션을 비교합니다.
   - 무제한 AI 대화 연습, 산업별 특화 어휘, 원어민 피드백 기능이 포함된 비즈니스 플랜에 관심을 갖습니다.
   - FAQ 섹션에서 추가 정보를 확인합니다.

3. **결제 진행**
   - '비즈니스 플랜 시작하기' 버튼을 클릭합니다.
   - 연간 결제 옵션을 선택하여 20% 할인을 받습니다.
   - 신용카드 정보를 입력하고 결제를 완료합니다.

4. **프리미엄 기능 탐색**
   - 업그레이드 완료 후 새로운 프리미엄 대시보드에 접근합니다.
   - 산업별 특화 콘텐츠 라이브러리를 탐색합니다.
   - 마케팅 및 비즈니스 협상 관련 스페인어 모듈을 발견합니다.

5. **고급 학습 활동**
   - AI 원어민과의 실시간 대화 연습 세션을 예약합니다.
   - 마케팅 프레젠테이션에 사용할 스페인어 스크립트를 작성하고 전문적인 피드백을 받습니다.
   - 개인화된 발음 트레이닝 세션을 진행합니다.

## 시나리오 4: 다국어 학습자

### 사용자 프로필
- **이름**: 태현
- **나이**: 29세
- **직업**: 여행 블로거
- **목표**: 여행을 위한 여러 언어의 기본 회화 습득
- **사용 기간**: 6개월

### 사용자 여정

1. **다중 언어 설정**
   - 태현은 계정 설정에서 학습할 언어로 프랑스어, 이탈리아어, 스페인어를 추가합니다.
   - 각 언어별 학습 목표를 기본 생활 회화 수준으로 설정합니다.
   - 각 언어에 주당 학습 시간을 할당합니다.

2. **언어 전환 학습**
   - 대시보드에서 오늘의 학습 언어로 프랑스어를 선택합니다.
   - 카페에서 주문하기 관련 레슨을 완료합니다.
   - 한 세션이 끝난 후, 이탈리아어로 전환하여 유사한 상황의 표현을 학습합니다.

3. **통합 복습**
   - 주간 복습 세션에서 세 언어의 유사한 표현을 비교하는 특별 모듈을 이용합니다.
   - 언어 간 유사점과 차이점을 통해 더 효과적으로 기억합니다.
   - 공통 어원을 가진 단어들을 그룹화하여 학습합니다.

4. **여행 특화 모듈**
   - '여행자를 위한 필수 표현' 모듈을 발견하고 선택합니다.
   - 각 언어별로 여행 중 필요한 상황별 표현을 학습합니다 (교통, 숙박, 응급 상황 등).
   - 가상 여행 시나리오를 통해 배운 표현을 실제 상황에서 연습합니다.

5. **성과 확인**
   - 언어별 진행 상황을 비교하는 통합 대시보드를 확인합니다.
   - 프랑스어 초급 수준 완료로 배지를 획득합니다.
   - 다음 여행 목적지인 스페인을 위해 스페인어 학습에 집중하기로 결정합니다.

## 시나리오 5: 학업 목적 사용자

### 사용자 프로필
- **이름**: 지원
- **나이**: 19세
- **직업**: 대학생
- **목표**: TOEFL 시험 고득점 획득
- **사용 기간**: 2개월

### 사용자 여정

1. **시험 특화 학습 경로 선택**
   - 지원은 학습 목표 설정에서 'TOEFL 시험 준비'를 선택합니다.
   - 목표 시험 일자와 목표 점수를 입력합니다.
   - 현재 영어 수준을 평가하는 모의 테스트를 완료합니다.

2. **맞춤형 학습 계획**
   - AI가 시험까지 남은 시간, 현재 수준, 목표 점수를 분석하여 학습 계획을 생성합니다.
   - 읽기, 듣기, 말하기, 쓰기 영역별 강점과 약점이 표시됩니다.
   - 약점으로 파악된 말하기와 쓰기 영역에 더 많은 시간이 할당됩니다.

3. **체계적 학습**
   - 매일 정해진 학습 모듈을 완료합니다.
   - TOEFL 읽기 지문 형식의 텍스트를 읽고 이해도를 테스트하는 문제를 풉니다.
   - 듣기 섹션에서 강의와 대화를 듣고 관련 질문에 답변합니다.

4. **시험 기술 습득**
   - 지원은 'TOEFL 시험 전략' 모듈을 학습합니다.
   - 시간 관리, 노트 테이킹, 답변 구조화 등 시험 기술을 습득합니다.
   - 실제 시험과 유사한 환경에서 모의 테스트를 진행합니다.

5. **성과 측정**
   - 주간 모의 테스트를 통해 점수 향상을 추적합니다.
   - 영역별 점수 변화를 그래프로 확인합니다.
   - AI가 분석한 개선 영역과 추가 학습이 필요한 부분을 확인합니다.

## 시나리오 6: 그룹 학습

### 사용자 프로필
- **그룹**: 회사 동료 5명
- **목표**: 해외 지사와의 협업을 위한 영어 비즈니스 커뮤니케이션 향상
- **사용 기간**: 신규

### 사용자 여정

1. **그룹 계정 설정**
   - 팀장이 기업용 그룹 계정을 생성합니다.
   - 5명의 팀원을 초대하고 각자 개인 프로필을 설정합니다.
   - 공통 학습 목표 (비즈니스 영어)와 기간 (3개월)을 설정합니다.

2. **그룹 평가**
   - 각 팀원이 초기 영어 수준 평가 테스트를 완료합니다.
   - 시스템이 그룹 전체의 평균 수준과 개인별 강점/약점을 분석합니다.
   - 팀 대시보드에 그룹 현황이 시각화됩니다.

3. **맞춤형 그룹 학습**
   - 공통 학습 모듈과 개인별 보충 학습이 할당됩니다.
   - 팀원들이 동일한 비즈니스 시나리오를 각자 학습합니다.
   - 그룹 채팅에서 학습 내용에 대해 토론하고 질문합니다.

4. **그룹 활동**
   - 주간 그룹 화상 회의에서 영어로 비즈니스 주제를 토론합니다.
   - AI가 대화를 분석하고 팀 전체에 피드백을 제공합니다.
   - 팀 프로젝트로 영어 비즈니스 제안서를 공동 작성합니다.

5. **성과 추적**
   - 그룹 리더는 전체 팀의 진행 상황을 관리 대시보드에서 모니터링합니다.
   - 개인별 참여도와 향상도가 그래프로 표시됩니다.
   - 월간 성과 보고서가 자동으로 생성되어 학습 효과를 측정합니다.

---

이러한 시나리오는 LangUp.ai가 다양한 사용자 요구를 어떻게 충족시키는지 보여줍니다. 각 사용자 여정은 서비스의 핵심 기능과 가치 제안을 강조하며, 사용자 경험 설계와 기능 개발에 기준점을 제공합니다.
